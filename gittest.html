현재 test로 branch가 설정되어있음 "git branch main"을 입력하면 원래 작성했던 main branch로 돌아간다. "git branch branch명"을 입력하면 다시 해당 branch로 이동할 수 있다. 코딩애플 행님께서 여러번 commit 해보라고 하셨으니 한 번 더 commit 해봄 merge 해봤는데 오류 떠서 다시 commit 해봄 숙제로 branch 새로 만들어서 다시 merge 해보라고 하셨는데 test 이거에 내용 다시 추가해서 merge 해 봄 잘 되네 ㅇㅇ -정리- 1. git branch 새로 만들고 싶다 = "git branch branch명" 2. branch 공간 바꾸고 싶다 = "git switch branch명" 3. 처음 작업했던 branch로 돌아가고 싶다 = "git switch main" 4. 특정 branch에서 작업한 소스코드가 마음에 들어서 main이랑 합치고 싶다 = "git merge branch명" (단, 모든 파일들이 commit 된 상태에서만 가능하다.) 5. merge 과정에서 코드들이 충돌된다면 직접 수동으로 수정해야 한다. * git squach 연습중 main branch에 commit 하나 남김.
