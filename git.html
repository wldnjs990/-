git 사용법 1. 인터넷에 git을 검색후 다운로드 한다.(다운로드 상세 설명은 유튜브 코딩애플 강의 참조) 2. 시작창에 터미널을 열어 "git --version"을 쳐서 깃이 깔렸는지 확인한다. 3. 깃을 사용할 대상 폴더에 powershell(그냥 터미널임, 우클릭해서 터미널 클릭)을 연다. 4. " git config --global user.email "이메일@naver.com" " 터미널에 다음과 같이 입력해 유저 이메일 등록을 한다. 5. " git config --global user.name "홍길동" " 터미널에 다음과 같이 입력해 유저 이름 등록을 한다. 6. 정보 입력을 완료했으면 애디터를 열고 해당 파일을 불러온 후에 "ctrl + shift + `"단축키로 애디터 터미널을 연다. 7. "git init"를 애디터 터미널에 입력해 해당 파일이 git의 영향을 받게 만든다. 8. "git add 파일명" 을 사용하면 코딩을 한 후에 코딩한 파일들을 staging area에 저장할 수 있다.(staging area == 작업폴더와 commit 사이의 중간다리 역할) 9. " git commit -m '메모할 내용' " 을 사용하면 stating area에 저장된 파일들을 commit 할 수 있다. * "git status"는 내가 보낸 staging area에 있는 코딩문서가 무엇이 있는지 확인 할 수 있다. * "git log --all --online"는 내가 commit한 코딩문서 내역을 보여준다.
